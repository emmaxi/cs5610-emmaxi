How to use Ajax to update a part of the page?

This is the part of the page to be changed bt AJAX

The purpose of this experiment is to used AJAX to update a part of the page. Using Ajax technonlogy, we can update the small part of our page without reload the whole one. AJAX means: Asynchronous JavaScript and XML. It works by sending the XMLHttpRequest to server and getting the response. 

           function loadXMLDoc() {
             var xmlhttp;
             xmlhttp = new XMLHttpRequest();

             xmlhttp.onreadystatechange = function () {
               if (xmlhttp.readyState == 4 && xmlhttp.status == 200) {
               document.getElementById("effectZone").innerHTML = xmlhttp.responseText;
             }
          }
          xmlhttp.open("GET", "ajax1.txt", true);
          xmlhttp.send();
         }

When the server response is ready, information embeded in our text file will be rendered in the arranged postion. When the user click the button "Change the text above", the whole precess are executed and render the data we got.

            <div id="myDiv">
                <h3 id="effectZone" style="color:Olive">This is the part of the page to be changed bt AJAX</h3>
            </div>
            <button class="btn btn-success" type="button" onclick="loadXMLDoc()">Change the text above</button>
