How to render charts using Google Charts API?

The purpose of this experiment is to explore and learn how to use the different types Google's Chart visualisation API and demonstrate them with an example

For the purpose of this experiment, we take the population statistics of the states Arizona and Massachusetts from the year 2000 to 2009.

To start with, We create three div containers in our page to hold 3 types of charts. 

            <div id="population_bar_chart"></div>
            <div id="population_line_chart"></div>

In order to use the google charts, we need the Google JSAPI API and the Google visualization library. We load the reletive supporting files into our page by using the below lines.

                <script type="text/javascript" src="https://www.google.com/jsapi"></script>
                <script type="text/javascript">
                             google.load("visualization", "1", { packages: ["corechart"] });

Next, we set a callback to run when the Google visualization API is loaded.

                google.setOnLoadCallback(drawChart);

Now, We prepare the data that needs to be plotted in the charts upon execution of the above call back function. 

          function drawChart() {
            var data = google.visualization.arrayToDataTable([
                ['Year', 'Arizona Population', 'Massachusetts Population'],
                ['2000', 5166697, 6363015],
                ['2001', 5304417, 6411730],
                ['2002', 5452108, 6440978],
                ['2003', 5591206, 6451637],
                ['2004', 5759425, 6451279],
                ['2005', 5974834, 6453031],
                ['2006', 6192100, 6466399],
                ['2007', 6362241, 6499275],
                ['2008', 6499377, 6543595],
                ['2009', 6595778, 6593587]
            ]);

            var options = {
                title: 'Population: Arizona Vs Massachusetts',
                legend: {
                    position: 'top'
                }

Now that we have all the data and the libraries in place, we finally call the class that draws that particular type of chart and pass the data, its options and the place holder on the page to which the charts needs to bind. 


            // For Bar Chart
            var chart = new google.visualization.BarChart(document.getElementById('population_bar_chart'));
            chart.draw(data, options);

                // For Line Chart
            var chart = new google.visualization.LineChart(document.getElementById('population_line_chart'));
            chart.draw(data, options);
