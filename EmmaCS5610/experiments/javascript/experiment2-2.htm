How to create animation using JQuery?

Click on the button, and observe what happens.

Clicking the button would smoothly change the dimentions of the rectangle. Show time~


 
 
 
 

core code and explain

This animation is a basic on which use the function of animate() in JQuery.In our demo, it shows four consecutive animation. The position, height and width of the default rectangle are change.

<script src="http://ajax.googleapis.com/ajax/libs/jquery/1.11.1/jquery.min.js"></script>

<script> $(document).ready(function () { $("button").click(function () { var div = $("#animatedArea"); div.animate({ left:'250px', height: '400px', opacity: '0.4' }, "slow"); div.animate({ left: '250px', width: '400px', opacity: '0.8' }, "slow"); div.animate({ left: '250px', height: '100px', opacity: '0.4' }, "slow"); div.animate({ left: '0px', width: '100px', opacity: '0.8' }, "slow"); }); }); </script>

<div id="animatedArea" style="background-color:Olive; height:100px;width:100px; position:relative;">
