core code and explain

The List shown before is a set of headers. Each header is clickable

Clicking a header will show relative information to that header

While the data for that clicked-header is shown, the data analogous to other headers are hidden.

These widgets can be used to structure data in our website, to specify information more clearly.

<link rel="stylesheet" href="//code.jquery.com/ui/1.11.2/themes/smoothness/jquery-ui.css"> <script src="//code.jquery.com/jquery-1.10.2.js"></script> <script src="//code.jquery.com/ui/1.11.2/jquery-ui.js"></script>
<script> $(function () { $("#accordion").accordion(); }); </script>
