How to make image slider using JQuery

using jQuery Plugin : jslider.js to show the image slider.


core code and explain

<script type="text/javascript"> $(function () { $(".slider").jslider({ btnNext: ".next", btnPrev: ".prev" }) }); </script>

<div class = "slider-container"> <img src="../../images/story_utility/arrowL.jpg" alt = "prev" class = "prev"/> <img src="../../images/story_utility/arrowR.jpg" alt = "next" class = "next"/> <div class = "slider"> <ul> <li><img src = "../../images/picture/slide2.jpg" alt = "watermelon"/></li> <li><img src = "../../images/picture/star.jpg" alt = "star"/></li> <li><img src = "../../images/picture/slide1.jpg" alt = "sun"/></li> </ul> </div> </div>

.prev { cursor:pointer; z-index:10; position:absolute; top:85px; left:0px; visibility:visible; }    same with the next button

explain:

z-index of 10 which is the highest to make sure that the button will show at the top.

cursor:pointer; cursor will change to a hand
