How to resize image using JavaScript and HTML5?

Silde the switch to zoom the image:
0%

The purpose of this experiment is to write a simple javascript that enable users to resize the image into whatever percentage they want.Usually the zoom is from 0% to 100%.

To start with, we have an image whose orignal size is 200px and its id="demo". We also have a HTML5 range control (id="zoomSelector") which is used to get the percentage zoom from the user. The zoomSelector can swing between 0% to 100% with 1% increase on each step. The onchange event of the zoomSelector is set to trigger a javascript function sizechange().

The sizechange() function is responsible for changing the size of the image as every input value. We start by declaring the original size of the image to keep the logic simple. 

                    Original height of the image in pixels:
                    var img_height = 200;
We then get the current zoom value provided by the user by using document.getElementById() function. 
                    Get the current zoom value of the range control:
                    var step = document.getElementById('zoomSelector').valueAsNumber;
Then, we calculate the height after changed. 
                    Calculate the percentage increase/decrease:
                    var changedHeight = img_height * (1+ (step / 100));

                    Set the new height to the image tag:
                    document.getElementById('demo').height = changedHeight;
Finally,We display the current zoom size of the image in percentage to the user 
                    Display the current zoom value to the user
                    document.getElementById('currentSize').innerText = step + '%';
