How to make the animation of search recommendation


In this experiment, after we type in some characters, the list of search recommendation will filter the results with an animation

  • {{name}}

core code and explain

To build this experiment, I add the abgular-animated.js from GOOGLE API.

This animation of search recommendation will show if the user input something in the search box. According to there imputs, the recommendation changes. For example, if the user input "w", then the recommendation will be "w","we","web" and "web developer". If the user input "we", then the recommendation will be "we","web" and "web developer"

Also, I used the Bootstrap to style the elements.
The animation is implemented by CSS3 animation.

<script src="http://ajax.googleapis.com/ajax/libs/angularjs/1.2.15/angular.min.js"></script>
<script src="//ajax.googleapis.com/ajax/libs/angularjs/1.2.15/angular-animate.js"></script>
<link href="//netdna.bootstrapcdn.com/twitter-bootstrap/2.3.1/css/bootstrap-combined.min.css" rel="stylesheet"/>

ng-app clearify the domain of AngularJS. ng-init allows us to set the initial variable values ​​before the program runs

<div class="showcase" ng-app="recommendation" ng-init="names=['w', 'we', 'web', 'web developer'];">

The angular.module() is a global place for creating, registering and retrieving Angular modules.When passed two or more arguments, a new module is created. If passed only one argument, an existing module (the name passed as the first argument to module) is retrieved

ngAnimate is used to create animation

<script>
angular.module('recommendation', ['ngAnimate']);
</script>

The ngRepeat directive instantiates a template once per item from a collection. Each template instance gets its own scope, where the given loop variable is set to the current collection item, and $index is set to the item index or key.

<li class="repeat-animation" ng-repeat="name in names | filter:search">
<a href="../../Default.aspx" style="color: #aaaaaa;">{{name}}</a>
</li>
