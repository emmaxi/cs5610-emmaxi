In this experiment, I realized a simple navigator on home page of my project.

This simple animation used ccs3. Fistly, I render the text.Then ,the heart actually is 2 parts.And I using position to combined them together. Then there is a hover effect,when hover happens, the arrow comes form the left side.

      .ph-heart-nav .effect {
      position: absolute;
      bottom: 26px;
      background-position: 0 0;
      height: 8px;
      width: 62px; 
      margin-left:-31px; }

    .ph-heart-nav a:before {
      content: "";
      display: block;
      position: absolute;
      left: 50%;
      bottom: 20px;
      background-position: -62px 0;
      height: 20px;
      width: 11px;
      margin-left: -11px; }

And every node has different postion for arrow to stop.So use the code below to figure it out

    .nav a:nth-child(1):hover ~ .effect {
	  left: 12.5%; }
	.nav a:nth-child(2):hover ~ .effect {
	  left: 37.5%; }
	.nav a:nth-child(3):hover ~ .effect {
	  left: 62.5%; }
	.nav a:nth-child(4):hover ~ .effect {
	  left: 87.5%; }

What's more, the arrow has it's own div tag and postion,then it can flow and ease-in-out. We must keep this div tag there.

    <div class="effect"></div>

     -webkit-transition: 0.7s ease-in-out;
     -moz-transition: 0.7s ease-in-out;
    -o-transition: 0.7s ease-in-out;
    -ms-transition: 0.7s ease-in-out;
    transition: 0.7s ease-in-out;
