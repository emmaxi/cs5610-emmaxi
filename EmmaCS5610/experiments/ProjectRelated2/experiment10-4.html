It's the alternative login page for my project.I realized the respnsive design using @media way.It split the whole page into 3 parts. When the max widht of window is 1440px and 1366px and the original width.

    /*-----start-responsive-design------*/
@media (max-width:1440px){
	.login-form {
	width:30%;
	margin: 11% auto 4% auto;
	}
	.main h1 {
	padding-top: 23%;
	}
}
@media (max-width:1366px){
	.login-form {
	width: 32%;
	margin: 10% auto 4% auto;
	}
}
