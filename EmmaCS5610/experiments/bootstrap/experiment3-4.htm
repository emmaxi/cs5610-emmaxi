How to use Bootstrap to embed video and show the progress bar?


the embeded video will load itself, as the progress bar shows. when the page is ready, a window with successful information will popup

when the animation of progress bar come to the end, the video will show on the page.

80% Complete

core code and explain

first of all, import all necessary links for Bootstrap and JQuery

second, crate a class named "progress" and then set the relative attributes. The special thing is that I use the class of "sr-only".

when the loading is finished,embeded video will show on the page and allow the full screen pattern.

use the embed responsive pattern for video,the embed address of the video will generate by Youtube automatically

<div class="progress-bar progress-bar-danger" role="progressbar" aria-valuenow="0" aria-valuemin="0" aria-valuemax="100" style="width:0%"> <span class="sr-only">80% Complete</span> </div>

<div class="embed-responsive embed-responsive-16by9"> <iframe width="560" height="315" src="https://www.youtube.com/embed/u5KQQWlIgGc" frameborder="0" allowfullscreen></iframe> </div>

Besides,We can also set the loading speed by ourself in JQuery

var interval = setInterval(function () {
widthNum += 10;
aria_valuenow += 10;
if (widthNum > 100) {
clearInterval(interval);
alert("Finished loading!")
videoDiv.show();
}
var width = widthNum + '%';
progressBar.css('width', width);
progressBar.attr("aria-valuenow", aria_valuenow);
}, 500);

here we set it as 500
