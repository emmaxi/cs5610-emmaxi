core code and explain

use these links below to import Bootstrap

<link rel="stylesheet" href="//netdna.bootstrapcdn.com/bootstrap/3.0.3/css/bootstrap.min.css">
<link rel="stylesheet" href="//netdna.bootstrapcdn.com/bootstrap/3.0.3/css/bootstrap-theme.min.css">
<script src="//netdna.bootstrapcdn.com/bootstrap/3.0.3/js/bootstrap.min.js"></script>

we create a Navigation,a Content area and a Footer in this page. The tag of <div class="jumbotron">,<div class="container">and some others are redefined in Bootstrap to make it more customer firendly.

Beside, uses grid system that utilizes a 12 column layout, meaning that the page can be divided by 12 equal columns if you so desire.

<div class="row">
<div class="col-md-6"> <h2>Emma</h2> <p><a class="btn btn-default" href="#" role="button">View details &raquo;</a></p> </div>
<div class="col-md-6"> <h2>Xi</h2> <p><a class="btn btn-default" href="#" role="button">View details &raquo;</a></p> </div>
</div>

you can see classes col-md-6. This will divide the page into two 6 columns width, making our content area into two equal sides. A benefit to using this grid system is that it works out the width of each of these colums relative to the screen size, so you don’t have to worry about writing any extra media queries to get the site working on all screen resolutions.
