How to use Bootstrap inputs and icons which is called Glyphicons?

Bootstrap form layouts: Vertical & Inline

Bootstrap can provide styles for form. By adding certain class names to <div> or <input>, we can get great style for the forms. The default one is a verical form. That means every element will be separate lines. I also try to use the inline style for the same form. When the screen goes beyond 768px wide, the two input fields will be listed in one line.

HTML Source code
<form role="form">
<div class="form-group"> <label for="email">Email Address:</label>
<input type="email" class="form-control" />
</div>
<div class="form-group">
<label for="comment">Comment:</label>
<textarea class="form-control" placeholder="Type your comment here."></textarea>
</div>
</form>

Bootstrap Icons: Glyphicons

Bootstrap has a collection of icons. After adding the icon files into the project, it's simply to add a built-in icon with a class name, such as glyphicon-ok. Demo for several inputs is shown as below.

Bootstrap comes with all 160 of Glyphicons Halflings set, all available in font formats for easy coloring, sizing, and placement.
